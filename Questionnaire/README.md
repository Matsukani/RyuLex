# 琉球諸語語彙用例調査票（RyuLex）

本フォルダには、琉球諸語の語彙および用例を体系的に収集するための調査票を収録している。

## 概要

本調査票は、語彙項目と例文を組み合わせた形で、自然な言語使用に基づくデータの収集を目的として作成されたものである。  

## 使用方法

- 各項目について、対象方言における自然な対応表現を収集する。
- 必要に応じて、文脈や語形を調整しながら記録する。
- 調査順は目安であり、状況に応じて柔軟に運用してよい。

## 例文について

例文は参考として提示されているものであり、調査時には適宜調整して使用することを想定している。  
例文1は松倉昂平氏作成の例文に基づき、同氏のご厚意により使用している。  
例文2は本調査票作成時に新たに作成したものである。

## 構成について

本調査票は、平山輝男（1992）『現代日本語方言大辞典』（明治書院）所収の「全国方言基礎語彙調査項目」を出発点としつつ、  
構成の再編成、項目の追加・削除、注記の付与、および用例文の新規作成を通じて再構成したものである。  
一部の項目については、国立国語研究所（2016）による「基礎語彙調査票」も参考にしている。

## 同源語ID（UniCog）

一部の項目には、UniCogに基づく同源語IDおよびアクセント類を付与している。  

これらは調査時の参考情報として用いることができる。
